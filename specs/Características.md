# Seguridad
- Seguridad y conexión correcta con la base de datos
- Validación de credenciales
- Controles de acceso
# Roles
El flujo de la app cambia según el rol, se deben respetar los roles. 
# Usabilidad
- Feedback: Mensajes al usuario a lo largo de toda la app. 
- Menú: Menú en todas las pantallas (Perfil, Home, Historial)
	- **Desktop:** Sidebar, puede ocultarse o desplegarse. 
	- **Mobile:** Tab bar, siempre visible.
- Persistencia de la sesión
- Permite el cerrado de sesión
